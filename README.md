# MFRMMx
Bayesian repulsive mixture model to cluster multivariate functional data

# Install
To install `MFRMMx`, consider the code
```R
install.packages("devtools")
devtools::install_github("rcpedroso/MFRMMx")
```

If installation fails, please report the problem to ricardocunhap@gmail.com. Any other problems or comments can also be reported!
